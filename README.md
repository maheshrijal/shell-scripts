## shell-scripts

### Install the latest version of Ansible

```
wget -O - https://maheshrjl.me/ansibleinstall | bash
```

### Install a specific version of Ansible

```
wget -O - https://maheshrjl.me/ansibleinstall | bash -s 2.10.0
```
